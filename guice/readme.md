# Hello World Guice

Basic hello world Guice.

1. Generate project

```
mvn archetype:generate -DgroupId=com.mycompany.app -DartifactId=guice -DarchetypeArtifactId=maven-archetype-quickstart -DarchetypeVersion=1.4 -DinteractiveMode=false
```

2. Build

```
mvn package
```


3. Trigger class

